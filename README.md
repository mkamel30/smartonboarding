# Smart Onboarding System

نظام تهيئة التجار الموحد - منصة متكاملة لإدارة طلبات التجار الجدد، متابعة الموافقات، وتخزين المستندات.

## 📖 التوثيق باللغة العربية (Documentation)

لقد تم إعداد توثيق شامل للنظام باللغة العربية لتسهيل الاستخدام والإدارة:

*   📘 **[دليل المستخدم (User Manual)](./docs/USER_MANUAL_AR.md)**: شرح كامل لواجهة البرنامج، الأدوار المختلفة (مدير، مشرف، مبيعات)، وكيفية إدارة الطلبات.
*   🛠️ **[الدليل الفني (Technical Guide)](./docs/TECHNICAL_GUIDE_AR.md)**: دليل تقني يشرح كيفية تشغيل البرنامج، استراتيجية النشر (Deployment)، وأوامر الصيانة.
*   🔄 **[دورة العمل الجديدة (Workflow V2)](./docs/WORKFLOW_V2.md)**: شرح الدورة الجديدة ذات الـ 7 مراحل لإدارة التهيئة.

## 🚀 تقنيات النظام (Tech Stack)

*   **Frontend**: React + Vite + Tailwind CSS
*   **Backend**: Node.js + Express
*   **Database**: PostgreSQL + Prisma ORM
*   **Deployment**: PM2 + Nginx
